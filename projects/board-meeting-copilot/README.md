# Board Meeting Copilot

A lightweight AI tool to turn raw meeting notes into structured board summaries.

## Purpose
Executives often need concise summaries of board meetings.  
This tool converts notes or bullet points into a structured overview.

## Example Inputs
- Budget increase approved
- Marketing campaign update
- Vendor follow-ups

## Output
- Key Decisions
- Action Items
- Discussion Highlights

## Built With
- Python
- Streamlit

## Future Enhancements
- AI summarization of notes
- Highlighting action items automatically
- Export PDF summaries
