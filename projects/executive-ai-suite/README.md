# Executive AI Suite



## Overview

The Executive AI Suite is a multi-tool AI application designed to support senior leaders in real-time decision-making. It combines multiple copilots into a single interface, enabling executives to summarise information, interpret performance metrics, and make structured decisions during meetings.



## Features

- Inbox Copilot: Summarises emails and suggests responses

- KPI Interpreter: Analyses business performance metrics and provides insights

- Decision Copilot: Structures decisions and recommends next steps

- Simple interface designed for fast use in meetings

- Built using Streamlit for rapid deployment



## How to Run

streamlit run app.py
